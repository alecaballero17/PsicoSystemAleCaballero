# 💻 Guía de Integración Web - PsicoSystem

## 📌 Módulo de Administración
- Uso del rol ADMIN para acceso a la configuración de la clínica y gestión de usuarios.
- **Onboarding:** El flujo de alta de clínica y administrador ya está atomizado para facilitar el registro de nuevos clientes.

## 📌 Módulo de Evolución y Diagnóstico
- Al guardar una nota de evolución, el sistema habilita el botón de 'Análisis IA'.
- El resultado de la IA debe mostrarse de manera clara para ayudar al diagnóstico preliminar del psicólogo.

## 📌 Reportes de Audio
- El sistema genera reportes personalizados en formato MP3. La URL del archivo se encuentra en la respuesta del endpoint de reportes.
